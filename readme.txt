3️⃣ Employee Class (this Pointer in Method)

Implemented:

🧠 Key Concepts I Understood
this is a special pointer available inside member functions.
this stores the address of the current object.
this->variable accesses object data members.
Helps remove confusion between:
class variables
constructor parameters
Makes code more readable and professional.
